# PowerPoint Karaoke Generator

Generate random presentations from any topic for PowerPoint Karaoke sessions.

## Features

- Generate presentations from any topic via LLM
- Export to PDF via Marp
- Configurable slide count and language
- Multiple LLM providers supported

## Usage

```bash
npx karaoke "Quantenphysik"
npx karaoke "Blockchain" --slides 8 --lang en
```
